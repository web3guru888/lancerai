⚡ x402 Protocol

Machine-Payable Endpoints

HTTP 402 Payment Required — AI agents can discover and pay for these endpoints automatically. No API keys needed, just USDC.

How x402 Works

  1. 1 Agent discovers endpoint via /.well-known/llms.txt or agent registry
  2. 2 Sends request → gets HTTP 402 with payment details (amount, address, chain)
  3. 3 Pays USDC on Base → includes tx hash in X-Payment header
  4. 4 Endpoint verifies payment → returns result. Fully autonomous machine-to-machine commerce!
